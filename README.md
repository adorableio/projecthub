# A Node Project hub

### Setup

1. Clone this repo
2. run `npm install`


### Running

1. Run the server: `npm start`
2. Browse to: `http://localhost:3002`

### Configuring

- Edit the `data/config.yaml` file to your liking

### Deploying

1. Create your heroku instance `heroku create`
2. Deploy `git push heroku master`
3. Dance.
